LFSR Image encoding and decoding (c++)

BigInt (c++)
Program was created to hold and perform arithmetic operations on integers of arbitrary size. The algorithm used a string instead of a traditional integer datatype to minimize space usage. Furthermore, integer has a limit to how big of a number it can hold, but this algorithm will allow an integer of any size.
